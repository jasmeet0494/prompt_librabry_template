# 05 — Weekly Cadence Call Documentation Workflow

- **Business function**: Finance / Cross-functional Performance Review / Stakeholder Coordination
- **Trigger**: Weekly cadence call involving finance, sales, and delivery teams is completed
- **Prompts in this section**: P01

---

# Section Purpose

This section supports the weekly cadence call documentation workflow by helping finance teams convert raw meeting notes or transcripts into a structured summary with clear action items and follow-up points. In recurring cross-functional revenue review meetings, important updates, risks, and decisions can be missed if note-taking is inconsistent or incomplete. This prompt helps standardise meeting documentation, improve accountability, and reduce the manual effort required to prepare a management-ready summary after each call.
